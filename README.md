# Advanced Calculator

## Overview

This is a Java-based Advanced Calculator that performs various mathematical operations through a menu-driven console application.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Modulus
* Power Calculation
* Square Root
* Percentage Calculation
* Invalid Input Handling
* Menu-Driven Interface

## Technologies Used

* Java
* Scanner Class
* Math Class
* Loops
* Switch Case Statements

## How to Run

1. Compile the program:

javac Calculator.java

2. Run the program:

java Calculator

## Sample Operations

* Add Two Numbers
* Subtract Two Numbers
* Multiply Two Numbers
* Divide Two Numbers
* Find Modulus
* Calculate Power
* Find Square Root
* Calculate Percentage
